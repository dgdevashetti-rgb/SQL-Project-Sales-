# 📊 SQL Sales Analytics Project

## 📌 Project Overview

This project contains SQL queries developed to analyze sales performance using the AtliQ Hardware (Croma) sales database. The objective is to solve real-world business problems by extracting meaningful insights from sales data.

The project demonstrates SQL skills required for Data Analyst and MIS Executive roles, including data aggregation, filtering, joins, grouping, ranking, and reporting.

---

## 🛠️ Tools & Technologies
- SQL
- MySQL
- Sales Database (AtliQ Hardware)
- GitHub

- ## 📂 Project Files

| File Name | Description |
|-----------|-------------|
| Croma Monthly Sales Report.sql | Generates monthly sales performance report for Croma. |
| Croma Sales Report for Fiscal_year 2021.sql | Analyzes Croma sales during FY2021. |
| Croma Yearly Sales Report.sql | Generates yearly sales summary. |
| Global Contribution of Net Sales.sql | Calculates country-wise contribution to total net sales. |
| Market Badge 1.sql | Identifies market badge based on business logic. |
| Top 3 customer by products with division.sql | Finds the top 3 customers by product quantity within each division using window functions. |

## 📊 Business Problems Solved
### 1. Monthly Sales Analysis
- Analyze monthly net sales
- Identify sales trends

### 2. Fiscal Year Sales Report
- Calculate sales for FY2021
- Measure annual business performance

### 3. Yearly Sales Report
- Generate yearly revenue summary
- Compare annual sales performance

### 4. Global Net Sales Contribution
- Calculate contribution percentage by country
- Identify top-performing markets

 ### 5. Market Badge Classification
- Classify markets using predefined business rules
- Support business segmentation
